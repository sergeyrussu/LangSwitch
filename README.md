# LangSwitch

LangSwitch is a lightweight macOS utility for fast keyboard layout switching.
It helps you quickly change the input language while typing and displays the current language flag.

## Installation
1. Open the [latest Release](https://github.com/sergeyrussu/LangSwitch/releases/).
2. Download the ZIP file from the latest release.
3. Unzip the archive.
4. Move `LangSwitch.app` to `/Applications`.
5. Launch `LangSwitch.app`.

## First Launch
If macOS blocks the app:
1. Open **Settings**.
2. Go to **Privacy & Security**.
3. Find the warning for LangSwitch.
4. Allow launch (`Open Anyway` or the equivalent button in your system language).
